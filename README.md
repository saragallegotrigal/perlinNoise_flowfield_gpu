# Flow Field Simulation with Perlin Noise (CPU vs CUDA GPU)

## Descripción del proyecto:
Este proyecto implementa una simulación de partículas guiadas por un flow field dinámico generado mediante Perlin Noise 3D fractal. 

El objetivo principal es comparar el rendimiento y comportamiento entre:  
- Implementación en CPU
- Implementación acelerada con CUDA en GPU  

Las partículas se mueven siguiendo un campo vectorial generado a partir del ruido Perlin, produciendo patrones orgánicos similares a humo, fluidos o corrientes de viento.  

La visualización en este caso se realiza usando SFML.
